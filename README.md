# Clothing-Review
Sentiment Analysis and Recommender System based on Women E-commerce dataset from Kaggle

## 2 approaches:
(1) Finetune pretrained models
- DistilBERT
- RoBERTa [highest accuracy]

(2) custom models
- NN
- LSTM

### Weights of finetuned RoBERTa model can be found:
